Welcome to the Escape Game!

This is a game of communication. Here are the rules
  1. You can play with a maximum of 4 players in the Escape Game.
  2. You must be able to call each other using any method like zoom or via phone call but are not allowed to screen share.
  3. Each person must choose a different character. You will have different clues on each screen.
  4. There is usually a password for each page that you must input to continue on using clues from everyone.
  5. If playing with 2-3 players, have one or both people select two different characters and open each in a different tab, so all clues can be seen.
Work Together to Escape. Good Luck!
